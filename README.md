# Site CV

Ce projet consiste à créer un site web personnel sous forme de CV interactif, en utilisant HTML, CSS et JavaScript, avec un design inspiré d'un site existant (`Bpost`). L'objectif est de recréer ce site en adaptant son contenu avec celui de notre CV, tout en s'assurant qu'il soit responsive et fonctionnel sans JavaScript.

### Langages utilisés :

- **HTML** : Structuration du contenu
- **CSS** : Mise en forme du site (Flexbox, Grid, animations)
- **JavaScript** : Interactivité (Événements, formulaire dynamique)
