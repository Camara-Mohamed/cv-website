## À propos

Ce projet est un **site web personnel sous forme de CV interactif**, créé pendant ma formation à la **HEPL - Techniques Infographiques option Web**.

L'objectif était de recréer un design web professionnel en m'inspirant de la structure du site Bpost, tout en le
personnalisant avec mon profil. Le site est **responsive**, **fonctionnel sans JavaScript** (dégradation gracieuse).

---

## Technologies

| Technologie | Utilité                                              |
|------------|------------------------------------------------------|
| HTML5 | Structuration sémantique du contenu                  |
| CSS3 + SASS | Mise en forme (Flexbox, Grid, Animations, Variables) |
| JavaScript | Interactivité avec dégradation progressive           |

## Installation


### Étapes

1. Cloner le repository
   ```bash
   git clone https://github.com/Camara-Mohamed/cv-website.git
   cd cv-website
   ```

2. Installer les dépendances
   ```bash
   npm install
   ```

3. Lancer le développement (ouvrir 2 terminaux)

   Terminal 1 - Compilation automatique des fichiers
   ```bash
   npm run watch
   ```

   Terminal 2 - Serveur de développement local
   ```bash
   npm run serve
   ```

   Le site est accessible sur `http://localhost:3000`
